# BMW Sales SQL Analysis

This project analyzes BMW sales data (2018–2025) using SQL queries.

## 📊 Dataset
- BMW global sales dataset (CSV file)

## 🧠 SQL Concepts Used
- SELECT, WHERE, GROUP BY
- JOINS
- Aggregations (SUM, COUNT)
- CASE WHEN
- Window Functions (RANK, LAG)

## 📈 Analysis Performed
- Top selling models
- Year-over-year revenue growth
- Region-wise contribution
- Monthly trends

## 🚀 Tools Used
- SQL (MySQL)
- Power BI (for visualization)
